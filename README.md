# 🛒 Shop Bot

Telegram e-commerce bot — product catalog, shopping cart, and checkout with inline keyboards.

![Python](https://img.shields.io/badge/Python-3.8+-3776AB?style=flat&logo=python&logoColor=white)
![aiogram](https://img.shields.io/badge/aiogram-3.x-26A5E4?style=flat&logo=telegram&logoColor=white)
![License](https://img.shields.io/github/license/XayitovB/shop-bot?style=flat)

## Features
- Product catalog with real-time inventory tracking
- Shopping cart (add / remove / clear)
- Order confirmation flow with inline keyboards
- Built on modern aiogram 3 (Router, Magic Filters `F`, InlineKeyboardBuilder)

## Quick Start
```bash
cp .env.example .env
pip install -r requirements.txt
python bot.py
```

## Stack
`Python 3.8+` · `aiogram 3` · `async/await`
